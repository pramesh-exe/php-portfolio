This inline style takes precedent over all other style
declarations. Next in order of precedent is the emebeded

Selectors

CSS selectors are used to apply style. There is the element selector, class selector and id selector. All being well you have used an element selector to modify the font on this page by adding style to the body tag in you attached style sheet. Words in this sentence have been styled using a class selector applied to letter-spacing and the heading for this section with an id selector.
